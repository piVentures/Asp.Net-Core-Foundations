# AuthDemo.API

AuthDemo.API is a simple ASP.NET Core Web API project that demonstrates how **Authentication** and **Authorization** are implemented in ASP.NET Core.

## Purpose
This repository is created for learning and demonstration of:
- Authentication
- Authorization
- Role-based access control
- Claim-based authorization
- Protected API endpoints using `[Authorize]`

## Tech Stack
- ASP.NET Core Web API
- C#
- .NET

## Run Locally
```bash
dotnet restore
dotnet run
